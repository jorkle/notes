# keyboard shortcut reference

## content

### keyboard shortcuts
| shortcut       | action               |
| -------------- | -------------------- |
| <kbd>F9</kbd>  | insert link to note  |
| <kbd>F10</kbd> | create child note    |
| <kbd>F11</kbd> | create neighbor note |
| <kbd>F12</kbd> | open parent note     |
| <kbd>F1</kbd>  | open note (search)   |
| <kbd>F2</kbd>  | outdent note         |
